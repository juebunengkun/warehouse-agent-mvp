# 指标口径库

## 销售额

- 业务含义：支付成功订单的实付金额。
- 计算口径：`SUM(pay_amount)`。
- 过滤条件：订单状态为 `PAID` 或 `FINISHED`，排除无效订单。
- 建议字段名：`sales_amount`。

## 订单数

- 业务含义：支付成功的去重订单数量。
- 计算口径：`COUNT(DISTINCT order_id)`。
- 过滤条件：订单状态为 `PAID` 或 `FINISHED`。
- 建议字段名：`order_count`。

## 支付用户数

- 业务含义：发生支付成功行为的去重用户数量。
- 计算口径：`COUNT(DISTINCT user_id)`。
- 过滤条件：订单状态为 `PAID` 或 `FINISHED`。
- 建议字段名：`pay_user_count`。

## 客单价

- 业务含义：平均每笔订单贡献的支付金额。
- 计算口径：`销售额 / 订单数`。
- 除零规则：订单数为 0 时返回 0。
- 建议字段名：`avg_order_amount`。

## 转化率

- 业务含义：访问用户中最终完成支付的比例。
- 计算口径：`支付用户数 / 访问用户数`。
- 注意事项：需要同时具备访问明细和支付明细。
- 建议字段名：`conversion_rate`。
